P. 6, first item #2, "somewhat exotic use of a map container". Cross-reference types are not top level commands.

Change:

This requirement applies only to the top-level commands like b or c, not to the commands a, s, or r entered during spell-checking mode

To:

This requirement applies only to the top-level commands like b, c or x, not to the commands a, s, or r entered during spell-checking mode. It also does not apply to the cross-reference types appearing after the x command, like a, w, i.
