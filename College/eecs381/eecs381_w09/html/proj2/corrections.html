NOTICE:

This document is part of the project specifications.

Your solution must follow these corrections and clarifications.


P. 1. Under No Length Restrictions: Strings are required only for variable length input.

Change:

In Project 2, a String class will be used for all input, and ...

To:

In Project 2, a String class will be used for input of all input lines, words, and file names, and ...

Notes: This change is consistent with the later specifications such as P. 3, under Specific Requirements, item number #3. This means that single-character commands can be read into a simple char variable.


P. 4, Use of the Standard Library, item #1. You cannot use the algorithm components of the STL.

Change:

You may not use the Standard Library <string> class or any of the Standard Library (or "STL") containers in this project.

To:

You may not use the Standard Library <string> class or any of the Standard Library (or "STL") containers, algorithms, or iterators in this project.

Explanation:

This project is specified as if the STL did not exist to give you practice in the concepts involved. The next project will involve very thorough use of the STL containers, algorithms, and iterators.
